## 印象笔记

| 快捷键       | 功能               | 补充                                                         |
| ------------ | ------------------ | ------------------------------------------------------------ |
| ctrl+shift+w | 新建符号列表       | 符号列表后按tab键可变为下一级符号列表，按退格键可变为上一级，按回车取消当前级符号列表 |
| ctrl+shift+l | 新建代码块         |                                                              |
| F2           | 重复执行刚才的指令 |                                                              |
|              |                    | 思维导图中按回车新建同级节点，按tap新建子级节点              |
|              |                    |                                                              |
